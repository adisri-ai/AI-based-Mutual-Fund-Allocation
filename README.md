# AI-based-Mutual-Fund-Allocation
An AI-based mutual fund Screener that uses historical mutual fund data to display the most optimized portfolio based on returns over a period of 10 years.
It takes the user's risk-taking capacity into consideration while generating the optimized portfolio along with the capital allocation to each of the 
constituent funds and also show the performance of the portfolio in the last 10 years. Implemented using python programming and uses streamlit as the 
web framework.
**App Link** :  https://fundalloc-zqnebddzozzqjrjzxczv6z.streamlit.app/
